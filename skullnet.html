SkullNet

Classifying mammalian locomotor ecology from cranial morphology using depth map rendering and transfer learning.

Overview

Trained EfficientNetB0 via transfer learning on depth map renders of 26 mammal skulls sourced from MorphoSource. Each 3D mesh (PLY/STL/OBJ) was rendered from 4 standardized angles using orthographic projection, encoding cranial geometry as grayscale depth maps rather than texture or color. A 3-class locomotion classifier (aquatic, arboreal, terrestrial) was trained on 104 images with 80/20 train/test split. Training accuracy reached 72% — well above the 33% random baseline. Generalization was tested on species absent from the training set, with correct predictions on walrus (Odobenus rosmarus, aquatic) and bonobo (Pan paniscus, arboreal). Applied speculatively to extinct taxa including Triceratops horridus.

Approach

Rather than using color, texture, or photographic images, each skull was converted into a grayscale depth map — a rendering where pixel intensity encodes distance from a virtual camera. This strips out all surface detail and forces the model to learn from shape alone. Four canonical views (anterior, posterior, lateral left, lateral right) were rendered per specimen using orthographic projection to eliminate perspective distortion.

Data

26 species sourced from MorphoSource, spanning three locomotor categories: aquatic (e.g. Lutra lutra, Zalophus californicus), arboreal (e.g. Varecia variegata), and terrestrial (e.g. Lynx rufus). 4 renders per skull, 104 total images. 80/20 train/test split.

Model

EfficientNetB0 pretrained on ImageNet, fine-tuned as a 3-class classifier. Transfer learning compensates for the small dataset size. Training accuracy reached 72% — well above the 33% random baseline for a balanced 3-class problem.

Speculative Application

Locomotor ecology is not directly observable in extinct taxa, but cranial morphology is. After validating generalization on extant species absent from training, the model was applied speculatively to fossil specimens including Triceratops horridus — not as a scientific claim, but as a demonstration of what geometric shape alone can encode.
